The SmartCart Clustering System is an intelligent, unsupervised machine learning pipeline designed to group shopping behaviors and segment customers based on the multiple features .

Language: Python
 Libraries: pandas, numpy (Data Processing), scikit-learn (Modeling), matplotlib, seaborn (Data Visualization)Environment: Jupyter Notebook or any Python IDE (e.g., VS Code)

 Unsupervised Modeling: Utilizes algorithms like K-Means, Hierarchical Clustering - Alggromerative clustering to find natural clusters without predefined labels.

 Characterization of Clusters & cluster Summary to analyse the behaviour of differnet type of customer to targeted campaigns and personalized recommendations.
